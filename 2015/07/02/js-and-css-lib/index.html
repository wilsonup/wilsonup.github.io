js、css库集中营

js

Animsition
Animsition是一个轻量级jquery页面动画过渡效果，包含58种特效

jQuery.Transit
使用 CSS3 的新特性来实现过渡效果

autosize.js
我们定义textarea(多行文本框)时,一般我们定义了大小,超出文本框时会出现下拉条,
使用该插件,文本框的大小会随着我们输入的内容自动扩大

Chart.js
一个简单、面向对象、为设计者和开发者准备的图表绘制工具库

chartist.js
简单的响应式图表，可以作为前端图表生成器(可以设置svg动画)

echarts
纯Javascript的图表库，可以流畅的运行在PC和移动设备上，兼容当前绝大部分浏览器（IE6/7/8/9/10/11，chrome，firefox，Safari等）

CoffeeScript
一门编译到 JavaScript 的小巧语言

dropzone
文件拖拽上传插件（有酷炫的上传动画，不支持IE低版本）

fancybox
一款基于jquery开发的类Lightbox插件

Lightbox
图片展示插件

favico.js
一个非常小的JS库。能够实现用图片、视频或徽章代替网站的Favicons图标。

FitText.js
jQuery经典响应式标题文字插件

gremlins.js
基于 JavaScript 编写的 Monkey 测试库，支持 Node.js 平台和浏览器中使用。Gremlins.js 随机模拟用户操作：单击窗口中的任意位置，在表格中输入随机数据，或者将鼠标移动到特别的元素

Headroom.js
为页面顶部多留些空间。在不需要页头时将其隐藏

Html2canvas
通过JS就能截屏，这是Html2canvas为用户提供的一个便捷的神器，通过对DOM元素的信息提取渲染页面。

impress.js
代替PowerPoint制作PPT

Intro.js
网站功能操作分布引导插件，支持使用键盘的前后方向键导航，使用 Enter 和 ESC 键退出指南。支持 Chrome、Firefox、Safari 浏览器以及部分IE 浏览器。

jquery.form
让你轻松管理表单数据和进行表单提交

Parallax.js
简单，轻量级的的视差引擎，智能设备的方向作出反应。

jQuery.payment
一个通用库验证输入和格式化数字

jquery-scrollbar.js
美化滚动条

jQuery.validate
表单验证

json2
提供了json的序列化和反序列化方法

modernizr
帮助我们检测浏览器是否实现了某个feature

moment.js
moment.js不依赖任何第三方库，支持字符串、Date、时间戳以及数组等格式，格式化日期时间，计算相对时间，获取特定时间后的日期时间等等

photoswipe
是专为移动触摸设备设计的相册/画廊

jQuery lightGallery
JQuery lightGallery is a lightweight jQuery lightbox gallery for displaying image and video gallery

ResponsiveSlides.js
轻量级幻灯片演示插件

reveal.js
一个制作3D幻灯片效果的插件(可全屏)

Unslider.js
一个超小的 jQuery轮播(slider)插件

ScrollMagic
超炫的页面滚动交互效果

Snap.js
帮助生成Facebook和Path风格的移动设备侧栏菜单的js类库

Snap.svg
用来创建互动，在任何大小的屏幕上都会很好看、与分辨率无关的矢量图形

svg.js
一个轻量级的JavaScript库，允许你轻松操作SVG和定义动画。

audio.js
提供了一种更加灵活的方式来使用HTML5的<audio>标签，当浏览器不支持<audio>标签时，它将自动换成Flash来支持Mp3文件的播放。

html5media.js
让所有浏览器支持HTML5 video视频标签

Countable.js
实时统计文本框里面的字数段落和字符

GMaps.js
可以很容易地在你的应用程序中集成谷歌地图，并包含了多个选项，如地图、方向、信息框等等

hammer.js
一款开源的移动端脚本框架，他可以完美的实现在移端开发的大多数事件，如：点击、滑动、拖动、多点触控等

html5shiv.js
完美解决IE(IE6/IE7/IE8)不兼容HTML5标签

iscroll
实现移动端各种效果以及解决一下bug

jquery.dotdotdot
实现多行省略号

jquery.fullpage
全屏滚动插件

jquery.nicescroll
美化滚动条

jquery.placeholder
实现 HTML5 的 placeholder 属性行为

jquery.ZeroClipboard
实现网站内容复制粘贴

Nprogress
用于页面刚打开时的页面加载进度显示

Respond.js
让不懂爱的 IE6-8 支持 CSS3 Media Query

Swipe.js
移动WEB页面内容触摸滑动类库

Swiper.js
移动端网站的内容触摸滑动(滚屏操作)

Underscore.js
提供了几十种函数式编程的方法

jPage.js
分页插件

css

colors.css
为网站更好的配色

Normalize.css
一个可以定制的CSS文件，它让不同的浏览器在渲染网页元素的时候形式更统一

Animate.css
css3动画库

pure.css
一组轻量级、响应式的CSS模块
