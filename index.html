js、css库集中营

js

Animsition
Animsition是一个轻量级jquery页面动画过渡效果，包含58种特效

jQuery.Transit
使用 CSS3 的新特性来实现过渡效果

autosize.js
我们定义textarea(多行文本框)时,一般我们定义了大小,超出文本框时会出现下拉条,
使用该插件,文本框的大小会随着我们输入的内容自动扩大

Chart.js
一个简单、面向对象、为设计者和开发者准备的图表绘制工具库

chartist.js
简单的响应式图表，可以作为前端图表生成器(可以设置svg动画)

echarts
纯Javascript的图表库，可以流畅的运行在PC和移动设备上，兼容当前绝大部分浏览器（IE6/7/8/9/10/11，chrome，firefox，Safari等）

CoffeeScript
一门编译到 JavaScript 的小巧语言

dropzone
文件拖拽上传插件（有酷炫的上传动画，不支持IE低版本）

fancybox
一款基于jquery开发的类Lightbox插件

Lightbox
图片展示插件

favico.js
一个非常小的JS库。能够实现用图片、视频或徽章代替网站的Favicons图标。

FitText.js
jQuery经典响应式标题文字插件

gremlins.js
基于 JavaScript 编写的 Monkey 测试库，支持 Node.js 平台和浏览器中使用。Gremlins.js 随机模拟用户操作：单击窗口中的任意位置，在表格中输入随机数据，或者将鼠标移动到特别的元素
